# Assets

Esta pasta é destinada ao armazenamento de imagens e outros arquivos estáticos utilizados no README principal.

## Como usar

Você pode colocar suas imagens aqui (por exemplo: `profile-image.png`, `animated-gif.gif`, etc.) e referenciá-las no README principal usando:

```markdown
![Descrição da imagem](./assets/nome-da-imagem.png)
```

Ou para imagens maiores:

```markdown
<img src="./assets/minha-imagem.png" width="400" alt="Descrição" />
```
